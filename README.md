# WCFGeneraPresupuestoNom
Sistema Generador del Layout de Presupuestos
Este sistema es parte del SIF (Sistema de Integración Financiera), el cual es operado por el área de Presupuestos de la UDLAP.
Desde este sistema se genearan los leyauts de las nominas semanales, quincenales, finiquitos y nominas esdpeciales. 
El layout generado es importado a Workday.
